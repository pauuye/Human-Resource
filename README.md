**Human Resources Analytics Project (SQL, Excel, Power BI)**

This project analyzes an Employee Productivity & HR dataset using MySQL, Excel, and Power BI.

It is a beginner-friendly end-to-end HR Analytics project focused on:

Employee performance

Absences

Tenure

Hiring trends

Department insights

The project includes data cleaning, exploratory analysis, and dashboard creation across three tools.
